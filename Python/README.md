# 序列
## 列表（list）
    - 列表是py中的一个对象（object）
    - object对象就是在内存中专门存储数据的区域
    - 像js中基本类型 string None number bool 都只能存储一个单一的数值
    - list 列表，则可以保存多个有序的数据
    - list 是用来存储对象的 对象
    - 列表的使用：
        1  列表的创建
        2  列表中数据的操作
    
## 序列
通过索引来操作的有序的数据结构，都是序列。 <br >
序列分类:
    - 一种是可变序列
        > list 列表
    - 一种是不可变序列
        > str 字符串(在底层是一个字符一个字符保存的)
        > tuple 元组
<br >
list的所有操作和方法，对序列同样适用，通用。
